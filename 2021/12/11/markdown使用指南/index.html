markdown使用指南

1.markdown的基础格式

1.1换行

  • 通过结尾添加两个空格来形成段落，或者p标签

这个指南实在是太详细啦！

我要认真学习！！

  • 使用空行来进行换行 或者是<br>

1.2字体

  • *斜体文字* 或 _斜体文字_ 效果为 斜体
  • **粗体文字** 或__粗体文字__ 效果为 粗体
  • ***粗斜体文字*** 或 ___粗体文字___ 效果为 粗斜体

1.3分割线

  • 通过在一行中三个以上的星号，减号，底线来创建分隔线并不能有其他东西，也能在期间插入空格，效果如 ***

    上面的就是分隔线

1.4删除线

  • 文字两端用两个波浪线夹住就能添加删除线，如: ~~delete~~

    delete

1.5块引用

  • 使用>能创建块引用(其实就是缩进里的进)，可以嵌套使用，如:

一个>

两个>

  • 可以嵌套下面所述的清单

  • 在markdown提供的列表中似乎会将输入其中的值默认转义插入不会解析其中的语句，因此像想在列表中插入样式建议还是使用html的原生制表语法

1.7列表间添加元素

  • 要在保留列表连续性的同时在列表中添加另一个元素，请将该元素缩进四个空格或一个制表符
* this is first sentence
* this is second
    > this is insert one
* this is third

输出为以下结果

  • This is the first list item.

  • Here’s the second list item.

    A blockquote would look great below the second list item.

  • And here’s the third list item.

1.8代码块

  • 代码块通常缩进四个空格或一个制表符。当它们在列表中时，将它们缩进八个空格或两个选项卡如下图所示
1.  Open the file.
2.  Find the following code block on line 21:

        <html>
          <head>
            <title>Test</title>
          </head>

3.  Update the title to match the name of your website.

以下为输出结果

  1. Open the file.

  2. Find the following code block on line 21:

     <html>
   <head>
     <title>Test</title>
   </head>

  3. Update the title to match the name of your website.
